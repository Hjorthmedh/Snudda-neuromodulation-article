# Snudda neuromodulation with RxD

This repository contains the code

SBTab model converted from Snudda/examples/notebooks/neuromodulation/data/SBtab_D1  (Nair_2016_optimized)

Original model found in:

Santos JP, Pajo K, Trpevski D, Stepaniuk A, Eriksson O, Nair AG, Keller D, Hellgren Kotaleski J, Kramer A. A modular workflow for model building, analysis, and parameter estimation in systems biology and neuroscience. Neuroinformatics. 2022 Jan;20(1):241-59.

URL: https://link.springer.com/article/10.1007/s12021-021-09546-3

Data repository: https://github.com/jpgsantos/Model_Nair_2016/releases/tag/1.0

Our version has changed true -> 1, false -> 0 in a few columns, all other things are unchanged.
